## Core CS Fundamentals

### Explain the concept of inheritance and give a real-world example.
Interviewers expect a concrete example (e.g. Vehicle → Car/Bike) plus a mention of code reuse
benefits.

### What is the difference between compile-time and run-time polymorphism?
Cover method overloading vs overriding with short code-shaped examples.

### Explain the difference between a stack and a queue, with real-world use cases.
LIFO vs FIFO, plus one practical example each (undo functionality vs print queue).

### What is indexing in a database, and why does it improve performance?
Cover the trade-off: faster reads vs slower writes and extra storage.

### Explain the OSI model briefly.
Infosys interviewers often want just the 7 layers named with one-line purposes, not deep detail.

## Java / Programming

### What is the difference between method overloading and overriding?
Cover same-class-multiple-signatures vs subclass-redefinition, and static vs dynamic binding.

### Explain the `static` keyword in Java.
Cover static variables, methods, and blocks, and when you'd use each.

### What is exception handling, and why is it important?
Cover try/catch/finally, checked vs unchecked exceptions, and graceful failure handling.

### Explain multithreading and a scenario where it's useful.
Cover concurrent execution basics and a practical example like handling multiple client requests.

### What is the difference between an array and an ArrayList in Java?
Cover fixed vs dynamic sizing, and type flexibility (generics) considerations.

## Databases

### Write a query to find employees earning more than the average salary.
Expect a subquery-based solution using `AVG()`.

### Explain joins in SQL — INNER, LEFT, RIGHT, FULL.
Give a short example for each with two small sample tables.

### What is a stored procedure, and why would you use one?
Cover reusability, reduced network round-trips, and centralized business logic.

## Web / Digital

### What is the difference between GET and POST HTTP methods?
Cover idempotency, data visibility (query string vs body), and typical use cases.

### Explain the concept of MVC architecture.
Cover Model, View, Controller separation and why it improves maintainability.

### What is CSS specificity, and how is it calculated?
Cover the general hierarchy: inline > ID > class > element, with a simple example.

## Cloud (for cloud-track candidates)

### What is the difference between horizontal and vertical scaling?
Cover adding more machines vs increasing a single machine's resources, and trade-offs.

### Explain the shared responsibility model in cloud computing.
Cover what the cloud provider secures vs what the customer is responsible for.
