## Coding Questions

Cognizant's GenC coding bar is Easy-to-Medium; GenC Pro/Elevate and lateral technical roles rise
to Medium, occasionally Medium-Hard for stronger digital-engineering positions.

### Find the largest and smallest element in an array in a single pass.
- **Difficulty**: Easy
- **Topic**: Arrays
- Tests whether you avoid unnecessary sorting for a simple min/max scan.

### Check if a string is a rotation of another string.
- **Difficulty**: Easy-Medium
- **Topic**: Strings
- Expect the concatenation-trick follow-up (`s1+s1` contains `s2` if it's a rotation).

### Implement a stack using an array or linked list.
- **Difficulty**: Easy
- **Topic**: Stacks
- Classic fundamentals check — push, pop, peek, isEmpty.

### Find all pairs in an array with a given difference.
- **Difficulty**: Medium
- **Topic**: Arrays / HashMaps
- A HashMap/HashSet-based O(n) approach is the expected optimal answer.

### Write a SQL query to find the Nth highest salary in a table.
- **Difficulty**: Medium
- **Topic**: SQL
- Expect discussion of `LIMIT`/`OFFSET` vs `DENSE_RANK()` approaches and their trade-offs.

### Given a list of transactions, detect if any account balance goes negative.
- **Difficulty**: Medium
- **Topic**: Arrays / Simulation
- Framed in a BFSI context, reflecting Cognizant's domain focus — tests sequential processing
  logic.

### Reverse a linked list in groups of K.
- **Difficulty**: Medium-Hard
- **Topic**: Linked Lists
- More common in GenC Pro/Elevate and lateral technical rounds.

### Find the first and last occurrence of a target element in a sorted array.
- **Difficulty**: Medium
- **Topic**: Binary Search
- Expect a modified binary search approach rather than a linear scan.

### Count the number of vowels and consonants in a string.
- **Difficulty**: Easy
- **Topic**: Strings
- Simple warm-up commonly used in the GenC assessment itself.

### Implement a basic case-insensitive word frequency counter for a paragraph of text.
- **Difficulty**: Easy-Medium
- **Topic**: Strings / HashMaps
- Tests string normalization and HashMap usage together.
